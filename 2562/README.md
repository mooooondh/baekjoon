<p>

<p>
	예를 들어, 서로 다른 9개의 자연수</p>

<p>
	3, 29, 38, 12, 57, 74, 40, 85, 61</p>

<p>
	이 주어지면, 이들 중 최댓값은 85이고, 이 값은 8번째 수이다.</p>
